# ALX HTML/CSS Project: CSS Basic

This project demonstrates the use of CSS for styling HTML documents. It includes:
- Two HTML files: `index.html` and `tweets.html`.
- Two CSS files: `base.css` and `styles.css`.

## Features
- Flexbox layout for responsive design.
- Semantic HTML structure.
- Custom styles applied to enhance readability and presentation.

## Usage
Open `index.html` or `tweets.html` in a browser to view the styled pages.

## Requirements
- All files are W3C-compliant.
- Developed with pure HTML and CSS.
